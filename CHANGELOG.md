### Features

- **core:** Add app, API, and storage constants
- **core:** Add custom exceptions and failure handling
- **core:** Add context, string, and date extensions
- **core:** Add Dio API client with interceptors (auth, logging, retry, token refresh)
- **core:** Add GoRouter with authentication guards
- **core:** Add secure storage and local storage
- **core:** Add theme system with Material 3 (light/dark mode)
- **core:** Add validators, formatters, and logger utilities
- **auth:** Add login, register, and forgot password screens
- **auth:** Add authentication state management with Riverpod
- **home:** Add dashboard screen with stats and quick actions
- **settings:** Add settings screen with theme toggle
- **onboarding:** Add 3-page onboarding carousel
- **shared:** Add primary button, text input, and card widgets
- **i18n:** Add English and Spanish localizations
